二、JS部分

依赖于jquery，需在有jquery的环境下使用；

条件触发时才会进行验证，如失去焦点，表单提交时进行验证，可自行定义；

示例如下：

	$('body').on('blur', 'input,textarea', function(){
		//统一验证的设置
		var obj = {
			inputObj : $(this)
		};
		//特殊设置
		switch($(this).name){
			case 'email' : 
				obj.sreg = validateEvn.rex.emailSreg;
				obj.reg = validateEvn.rex.email;
				break;
		}
		//调用
		validateEvn.validate(obj,function(opt){
			opt.inObj.parent().find('.tips').html(opt.tips);
		});
	});
